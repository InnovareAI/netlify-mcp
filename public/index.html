This is a Model Context Protocol (MCP) server for Netlify integration. It provides tools to manage Netlify sites programmatically.

Available Tools:

  • createSiteFromGitHub - Create a new site from a GitHub repository
  • listSites - List all your Netlify sites
  • getSite - Get details of a specific site
  • deleteSite - Delete a site
  • createEmptySite - Create an empty site for manual deployment

Installation:

npm install -g netlify-mcp-server

Usage:

netlify-mcp
